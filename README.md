## Getting Started

First Clone The Repo By Click On The Code And HTTPS or SSH Copy The Link:
Then Run Open Terminal And Run The Command
```bash
git clone paste the copied link
```
Then
```bash
npm install
npm run dev
# or
yarn dev
# or
pnpm dev
# or

```
